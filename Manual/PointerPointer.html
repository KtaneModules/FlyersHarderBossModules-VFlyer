On the Subject of Pointer Pointer

Where are you pointing at?

The module consists of a 6 by 6 grid filled with arrows in different colors/orientation.

In the beginning of the module, a single arrow will light up. This will be your starting position. Write down all of the orientation/color of the arrows. After a module is solved, all arrows from the illustration will be removed. Using the illustration from the arrows previously, move until you encounter a tile twice, looping to the other side if a pointer is not pointing on a tile. The tile that is encountered twice for the first time will be the correct tile. If the tile selected is correct, the lights beside the grid will flash green, and you would be able to continue to solve modules without problem. If the tile is incorrect, or a module is answered while the module is not answered, the light beside the display will flash red and a strike will occur.

After answering correctly, a new set of arrows with different colors/orientation and a new starting point will show. Use the same step that was performed in the beginning until all the  modules has been solved.

Note:

  • Some modules are ignored by the module.

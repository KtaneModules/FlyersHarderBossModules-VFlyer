Keep Talking and Nobody Explodes Mod Ten Digit Keypad

On the Subject of Ten Digit Keypad

Someone got lazy again and forgot to include the labels.

This module consists of an unlabeled 10-digit keypad arranged in a keypad-like formation. The digits on the keypad in reading order input the digits 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, respectively. In addition, there are 2 displays, a rotated wide display showing the last 3 correct inputs, and a display on the top-left showing a continuous sequence of digits. The digit shown from the top-left screen will be in yellow if this is the current position of the sequence to input, green if the digit in that position is already inputted, or white otherwise. When no correct digits are inputted, the first digit to input will be in yellow, and no green digits are displayed.

For each digit to input, start with the displayed digit. The digit to input will be displayed digit in that sequence modified by the first condition from the provided list, and kept within 0 - 9 by adding/subtracting 10.

First Digit to Input

  • If an FRK indicator is present, the modifier is +2 if that indicator is lit, -2 if unlit.
  • Otherwise, if there are no lit indicators, the modifier is -(number of unlit indicators).
  • Otherwise, if there are more lit indicators than unlit indicators, the modifier is -7.
  • Otherwise, the modifier is +(2nd digit of the serial number).

Second Digit to Input

  • If there is an empty port plate, the modifier is 0.
  • Otherwise, if the previous digit inputted is not divisible by 2, the modifier is +(least significant number of digits displayed, on the top-left)
  • Otherwise, the modifier is -(1st digit inputted - 1).

All Other Digits to Input

  • If at least one of the last 2 inputted values were a 0, the modifier is +(1st digit of the serial number)
  • Otherwise, if the 2nd to last inputted value is even and the last inputted value is odd, the modifier is +(sum of all odd digits in the serial number).
  • Otherwise, the modifier is +(left-most digit of the sum of the last 2 inputted values).
